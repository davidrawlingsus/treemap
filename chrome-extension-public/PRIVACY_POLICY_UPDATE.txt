Add new section 2.4 — Information Collected by the Chrome Extension:

2.4 Information Collected by the Chrome Extension

When you use the MapTheGap — Ad Intelligence Chrome extension, we collect and process the following data:

Email Address:
Your email address, provided when you register through the extension. Your email domain must match the advertiser's domain to verify your association with the business.

Advertiser Ad Data:
Ad copy, headlines, descriptions, call-to-action text, ad format, status, run dates, and destination URLs from the Meta Ads Library page you are viewing. This data is publicly available on Facebook's Ad Library.

Ad Analysis Results:
AI-generated analysis of your ad copy, including quality scores, creative assessments, and improvement recommendations. This analysis is generated using the Anthropic Claude API and stored in our database.

Review Platform Data:
When you grant permission, we access your website to detect review platforms (e.g., Trustpilot, Okendo, Yotpo) and analyze publicly available customer reviews. This data is used to compare your customer voice with your ad copy.

Usage Analytics:
Anonymous usage events (e.g., extension opened, analysis started, analysis completed) with a session identifier. These events do not contain personally identifiable information beyond what is already collected above.

Browser Permissions:
The extension requests optional access to your website domain to fetch page content for review platform detection. This permission is requested explicitly and can be denied without affecting core functionality.


Add new section 3.3 — Chrome Extension Data:
------------------------------------------------------------------------

3.3 Chrome Extension Data

We use data collected through the Chrome extension for:

Ad Copy Analysis:
Analyzing publicly available Facebook ad copy to provide quality scores, creative assessments, and actionable recommendations for improvement.

Review Signal Analysis:
Detecting and analyzing publicly available customer reviews to identify voice-of-customer insights that can strengthen your advertising.

Opportunity Assessment:
Calculating an opportunity score that compares the quality of your ad copy against the richness of your customer review data, identifying potential for improvement.

Account Creation:
Creating and managing your MapTheGap account, linking your analysis results to your business profile for future reference.

Service Improvement:
Analyzing anonymous usage patterns to improve the extension's functionality and user experience.


Add to Section 4.2 (Limited Sharing) — new bullet:
------------------------------------------------------------------------

AI Processing:
Ad copy and review text are sent to Anthropic's Claude API for analysis. Anthropic processes this data according to their privacy policy and does not use API inputs for training. No personally identifiable information is included in these API requests — only the ad copy text and review content.


Add new section — Chrome Extension Permissions:
------------------------------------------------------------------------

[NUMBER]. Chrome Extension Permissions

The MapTheGap Chrome extension requests the following browser permissions:

Required Permissions:
- activeTab: To interact with the Facebook Ads Library page you are viewing
- storage: To store your authentication token locally in your browser
- sidePanel: To display the analysis interface as a browser side panel

Required Host Permissions:
- facebook.com/ads/library: To extract publicly available ad data from the Meta Ads Library
- api.mapthegap.ai: To communicate with our analysis backend
- vizualizd.mapthegap.ai: To handle authentication

Optional Host Permissions:
- Your advertiser's website domain: Requested only when needed to detect review platforms. You can deny this permission without affecting ad analysis functionality. This permission is requested per-domain (e.g., "Read data on example.com"), not for all websites.

Data stored locally in your browser (via chrome.storage) includes only your authentication token. All analysis data is stored on our servers, not in your browser.
